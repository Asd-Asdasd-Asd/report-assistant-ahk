# 麦旋风配置指南

本指南用于修改快捷语、触发词和可选功能。普通使用不需要修改配置。

## 配置文件

配置文件位于：

```text
%LOCALAPPDATA%\MedExReportAssistant\config.ini
```

配置只在程序启动时读取。修改前先保存正在编辑的报告，保存配置文件后，右键系统托盘图标并选择“重新加载配置”。无需手工退出再重新打开。

文件编码必须保持为带 BOM 的 **UTF-16 LE**。建议使用 Windows 记事本编辑；另存时选择 `UTF-16 LE`。不要把配置文件移动到 EXE 同目录。

## 基本结构

```ini
[Config]
SchemaVersion=2

[Features]
GlobalHjklArrows=false

[Hotstring.builtin-red]
Enabled=true
Name=红字插入
Trigger=;red
Text={{red:（见图）}}
```

`SchemaVersion` 不要手动修改。新版本可能在创建备份后补充缺失的程序默认项，但不会覆盖已有配置值。

## 可选全局方向键

```ini
[Features]
GlobalHjklArrows=true
```

启用后：

| 按键 | 作用 |
| --- | --- |
| `RAlt+H` | Left |
| `RAlt+J` | Down |
| `RAlt+K` | Up |
| `RAlt+L` | Right |

该功能默认关闭并且作用于所有应用。启用前应退出仍注册相同按键的 legacy/compatibility script。

## Hotstring section

每项 section 必须命名为：

```text
Hotstring.builtin-<stable-id>
Hotstring.custom-<user-id>
```

支持字段：

| 字段 | 含义 |
| --- | --- |
| `Enabled` | `true` 注册，`false` 禁用 |
| `Name` | 便于识别的名称 |
| `Trigger` | 用户输入的完整触发词 |
| `Text` | 插入的文字 |

不要添加 `Mode`、`Order` 或 Left 数。模板行为由 `Text` 决定。

## 模板占位符

- `{{cursor}}`：插入后光标停留的位置；每个模板最多使用一次。
- `{{date}}`：触发时的当天日期，格式为 `yyyy-MM-dd`；可以使用多次。
- `{{red:（见图）}}`：插入红色 `（见图）`；每个模板最多使用一次，并且必须放在模板最后。

例如 `cm×{{cursor}}cm` 会插入 `cm×cm`，并把光标停在中间。普通单花括号可以正常作为正文。拼错、未知、未闭合的双花括号占位符会被设置窗口拒绝。

普通字面量 `（见图）` 始终按黑字处理。只有精确的 `{{red:（见图）}}` 会插入红字，不支持其他 `{{red:...}}`。光标停在模板内部时不会执行颜色恢复；光标位于红色尾标记之后时，程序才会恢复黑色。普通黑字模板不会调用颜色恢复。

## Text 转义

- 两个字符 `\n` 表示换行。
- 两个反斜杠 `\\` 表示一个普通反斜杠。
- 配置文字只作为文本发送，不会作为 AutoHotkey code 执行。

## 自定义示例

新生成的配置文件末尾包含一段默认关闭的示例。请复制完整内容，不要直接修改原始示例：

```ini
[Hotstring.custom-example]
Enabled=false
Name=新的快捷语
Trigger=;example
Text=请输入内容
```

复制后，必须把方括号里的 `example` 改成不重复的英文名称。这里只使用小写英文字母、数字和减号；不要使用中文、空格，也不要继续使用 `example`。`Name` 和 `Text` 可以填写中文。

例如：

```ini
[Hotstring.custom-lung-note]
Enabled=true
Name=肺部描述
Trigger=;lung
Text=请输入内容
```

旧 Schema 1 配置会在首次启动时自动审计并备份，只有能完整保留旧行为时才升级为 Schema 2。开发阶段已经生成的 Schema 2 配置中，仍与旧 builtin 默认文字完全一致的项目会自动换成新的模板元素；程序不会覆盖用户修改过的 builtin 或自定义模板。升级失败不会覆盖原文件。升级后如需使用旧 EXE，必须先恢复迁移前备份。

## 重复和无效配置

- 所有模板（包括停用模板）的 `Trigger` 都不能重复。
- 空名称、空 `Trigger`、无效 `Enabled` 或错误占位符不能保存。
- 文件不可读、schema 不支持或迁移不安全时，报告模板会停用，原文件不会被猜测性改写。
- 不要配置 position、coordinate、timing、offset 或 Left-count。

如果修改后行为异常，退出程序并恢复配置文件备份，或联系维护者。不要发送包含患者信息的配置示例、截图或日志。
