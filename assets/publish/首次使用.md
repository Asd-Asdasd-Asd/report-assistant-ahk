# 麦旋风首次使用

麦旋风是 MedEx Report Assistant 的内部测试版本，用于辅助输入常用报告文字。它不会替你审核或提交报告。

## 开始前

1. 将维护者提供的 ZIP 复制到本机磁盘。
2. 完整解压 ZIP；不要直接从共享盘或压缩包内运行。
3. 如果旧版正在运行，先在系统托盘退出旧版。
4. 双击 `麦旋风.exe`。

EXE 可以放在 Desktop、普通本地文件夹或 Windows Startup folder，不需要安装，也不要求管理员权限。

## 确认运行状态

启动后，系统托盘会出现 AutoHotkey 图标。如果看到“MedEx Report Assistant 已在运行”，请先从系统托盘退出当前版本，再启动新版本。

## 常用输入

在 MedEx 报告输入框中输入完整缩写：

| 输入 | 作用 |
| --- | --- |
| `;red` | 插入红色 `（见图）` |
| `;fzg` | 插入 SUVmax 相关文字，并把光标移到填写数值的位置 |
| `;fwj` | 插入“放射性摄取未见明显增高（见图）” |
| `;fjd` | 插入“放射性摄取降低（见图）” |
| `;cmx` | 插入 `cm×cm`，光标停在两个数值之间 |

## 暂停与退出

- `Ctrl+Alt+Esc`：暂停或恢复。
- `Ctrl+Alt+Q`：退出。
- 修改配置并保存后，右键系统托盘图标并选择“重新加载配置”。
- 也可以右键系统托盘图标并选择 Exit。

## 配置与更新

自定义配置保存在：

```text
%LOCALAPPDATA%\MedExReportAssistant\config.ini
```

删除或替换 EXE 不会删除该配置。修改快捷语、触发词或模板文字前，请阅读同目录的 `配置指南.md`。

更新方法：

1. 在系统托盘中退出旧版 MedEx Report Assistant。
2. 删除或覆盖原来的 `麦旋风.exe`。
3. 下载新版 ZIP，复制到本机并完整解压。
4. 运行新的 `麦旋风.exe`。

原有自定义配置保存在本机用户目录中，不会因删除 EXE 而丢失。

## 出现异常

先按 `Ctrl+Alt+Esc` 暂停；必要时按 `Ctrl+Alt+Q` 退出。检查报告中的可见文字、颜色和光标位置并手工修正，然后联系维护者。

不要发送患者信息、报告文字、剪贴板内容、账号或包含临床信息的截图。
