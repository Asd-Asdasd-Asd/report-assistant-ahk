# 麦旋风首次使用

写报告时那些反复要打的句子、要标红的"（见图）"、打完红字还得手动调回黑色——这些用几个缩写就能一次搞定。麦旋风只帮你输入，不会替你审核或提交报告。

## 开始前

1. 把我给你的 ZIP 复制到本机（别在压缩包里或共享盘上直接运行）。
2. 完整解压。
3. 如果旧版在运行，先从屏幕右下角的托盘图标退出它。
4. 双击「麦旋风.exe」。

不用安装，也不需要管理员权限。EXE 放桌面或本地文件夹都行；想开机自启就放进 Windows 启动文件夹。

## 确认在运行

启动后，屏幕右下角托盘会出现麦旋风图标。如果提示已在运行，先从托盘退出当前的，再启动新的。

- 双击图标：打开设置。
- 右键图标：设置、重新加载配置或退出。

## 常用输入

在 MedEx 报告框里打出完整缩写：

| 打这个 | 出来的内容 |
| --- | --- |
| ;red | 红色「（见图）」 |
| ;fzg | SUVmax 那句话，光标停在填数值的地方 |
| ;fwj | 放射性摄取未见明显增高（见图） |
| ;fjd | 放射性摄取降低（见图） |
| ;cmx | cm×cm，光标停在两个数值中间 |

想改这些内容、或加自己的，看同目录的「配置指南.md」。

## 暂停与退出

- Ctrl+Alt+Esc：暂停 / 恢复。
- Ctrl+Alt+Q：退出。
- 也可以右键托盘图标退出。

## 更新

新版怎么换，看「更新说明.md」。你的个人设置会自动保留，不用手工搬。

## 不对劲的时候

先按 Ctrl+Alt+Esc 暂停，必要时按 Ctrl+Alt+Q 退出，手动把报告里的文字、颜色、光标改好，然后来找我。先别自己反复删设置或重装。
