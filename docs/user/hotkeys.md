# 常用缩写和快捷键

在报告输入框里输入左侧内容，会出现右侧内容。

| 输入什么 | 出现什么 |
| --- | --- |
| `;red` | `（见图）` |
| `;fzg` | `放射性摄取增高，SUVmax约（见图）`，光标会回到适合填写数值的位置 |
| `;fwj` | `放射性摄取未见明显增高（见图）` |
| `;fjd` | `放射性摄取降低（见图）` |
| `;cmx` | `cm×cm`，光标会回到两个数字中间 |

## 全局快捷键

| 按什么 | 做什么 |
| --- | --- |
| Ctrl+Alt+Esc | 暂停或恢复工具 |
| Ctrl+Alt+Q | 退出工具 |

如果工具行为异常，先按 Ctrl+Alt+Esc 暂停。
