# 常用缩写和快捷键

在报告输入框里输入左侧完整缩写后，会自动出现右侧内容。不需要再按空格、回车或标点。

| 输入什么 | 出现什么 |
| --- | --- |
| `;red` | 插入红色 `（见图）` |
| `;fzg` | `放射性摄取增高，SUVmax约为` 后接红色 `（见图）`，光标会回到适合填写数值的位置 |
| `;fwj` | `放射性摄取未见明显增高` 后接红色 `（见图）` |
| `;fjd` | `放射性摄取降低` 后接红色 `（见图）` |
| `;cmx` | `cm×cm`，光标会回到两个数字中间 |

## 全局快捷键

| 按什么 | 做什么 |
| --- | --- |
| Ctrl+Alt+Esc | 暂停或恢复工具。暂停后再按一次可以恢复 |
| Ctrl+Alt+Q | 退出工具，即使工具已经暂停也可以退出 |

以下全局方向键默认关闭；确有需要时请联系维护者协助启用：

| 按什么 | 做什么 |
| --- | --- |
| RAlt+H | 向左 |
| RAlt+J | 向下 |
| RAlt+K | 向上 |
| RAlt+L | 向右 |

启用前请退出原始 `legacy/karabiner.ahk`。清理后的 `medex_legacy_compat.ahk` 已不再注册这四个方向键，可以与麦旋风同时运行。

如果工具行为异常，先按 Ctrl+Alt+Esc 暂停。

如果 `（见图）` 不是红色，请暂时不要依赖这个快捷输入，并联系维护者。正常情况下，不需要手动从 Word 复制红色文字。
