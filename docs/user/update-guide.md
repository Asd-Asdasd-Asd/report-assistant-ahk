# 麦旋风更新方法

麦旋风采用便携式发布文件夹。程序不要求固定安装目录，也不会自动下载、替换或清理其他版本。

## 更新方法

1. 在系统托盘中退出旧版麦旋风。
2. 下载新版 ZIP，复制到本机并完整解压。
3. 用新版完整发布文件夹替换旧程序文件。
4. 运行新的 `麦旋风.exe`。
5. 首次启动时，程序会自动处理旧配置。

不要再使用旧版 EXE 打开已经升级的配置。

个人配置保存在本机用户目录中，通常不需要手工复制、删除或编辑：

```text
%LOCALAPPDATA%\MedExReportAssistant\config.ini
```

## 下载位置

请先把维护者提供的 ZIP 复制到本机磁盘，再完整解压到 Desktop、普通本地文件夹或其他自选位置。不要直接从共享盘、共享驱动器或 ZIP 压缩包内部运行 EXE。

如果需要开机自动运行，可以由用户手工把 EXE 放入 Windows Startup folder；程序本身不会创建 Startup shortcut。

## 如果无法启动

如果看到麦旋风已在运行的提示，请先通过系统托盘退出当前版本，再启动新版本。程序不会自动终止或替换正在运行的版本。

不要删除或手动覆盖 `config.ini`。如需截图，请先遮住患者信息、报告文字、账号和其他敏感内容。
