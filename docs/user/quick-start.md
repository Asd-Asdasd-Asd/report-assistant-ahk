# 快速开始

这个工具用于辅助报告书写，帮助输入一些常用文字。它不会替你自动提交报告。

## 使用前准备

- 确认维护者已经给你准备好最新文件。
- 关闭旧版本工具。
- 先把 ZIP 复制到本机并完整解压，不要直接从共享盘运行。
- 打开报告书写窗口。

## 如何启动

双击维护者提供的 `麦旋风.exe`。EXE 可以放在 Desktop、普通本地文件夹或 Windows Startup folder，不要求固定安装位置。

## 如何确认正在运行

启动后，右下角托盘区域通常会出现麦旋风图标。如果另一个版本已经运行，新进程会提示先从系统托盘退出旧版本，并自行退出。

双击麦旋风图标可以打开设置。也可以右键图标选择“设置…”或“重新加载配置”。

## 最常用功能

在报告输入框里输入缩写，例如 `;cmx` 或 `;red`，工具会自动替换成对应内容。

常用缩写请看 `hotkeys.md`。

## 如何暂停

按 Ctrl+Alt+Esc。

暂停后，再按一次 Ctrl+Alt+Esc 可以恢复。

## 如何退出

按 Ctrl+Alt+Q。

也可以在右下角麦旋风图标上点右键，然后选择退出。

## 出问题怎么办

先按 Ctrl+Alt+Esc 暂停。如果仍然异常，按 Ctrl+Alt+Q 退出，然后联系维护者。
