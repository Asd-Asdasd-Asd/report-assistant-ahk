# 快速开始

这个工具用于辅助报告书写，帮助输入一些常用文字。它不会替你自动提交报告。

## 使用前准备

- 确认维护者已经给你准备好最新文件。
- 关闭旧版本工具。
- 打开报告书写窗口。

## 如何启动

双击维护者提供的 `report_assistant.ahk` 文件。

## 如何确认正在运行

启动后，屏幕上会短暂出现提示。右下角托盘区域通常也会出现绿色 `H` 图标。

## 最常用功能

在报告输入框里输入缩写，例如 `;cmx` 或 `;red`，工具会自动替换成对应内容。

常用缩写请看 `hotkeys.md`。

## 如何暂停

按 Ctrl+Alt+Esc。

暂停后，再按一次 Ctrl+Alt+Esc 可以恢复。

## 如何退出

按 Ctrl+Alt+Q。

也可以在右下角绿色 `H` 图标上点右键，然后选择退出。

## 出问题怎么办

先按 Ctrl+Alt+Esc 暂停。如果仍然异常，按 Ctrl+Alt+Q 退出，然后联系维护者。
