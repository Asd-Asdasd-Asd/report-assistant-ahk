# Calibration

Viewer coordinates depend on screen resolution, display scaling, window placement, application layout, and workstation configuration.

Do not reuse coordinate maps across workstations without testing.

## Calibration Notes

- Copy `src/config.example.ahk` to `src/config.local.ahk`.
- Adjust executable names if they differ locally.
- Record only generic coordinate names and numeric positions.
- Do not store screenshots, patient data, hospital identifiers, credentials, or sensitive logs.
- Test each coordinate action manually before enabling it in workflow code.
