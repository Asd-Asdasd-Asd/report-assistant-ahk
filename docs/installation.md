# Installation

## Requirements

- Windows workstation
- AutoHotkey v2
- Access to the local report-writing environment

## Steps

1. Install AutoHotkey v2 from the official AutoHotkey website.
2. Copy `release/report_assistant.ahk` to the target Windows workstation.
3. If running from source, copy `src/config.example.ahk` to `src/config.local.ahk` and calibrate local values.
4. Double-click the `.ahk` file or run it with AutoHotkey v2.
5. Complete the manual checklist in `tests/manual-test-checklist.md`.
